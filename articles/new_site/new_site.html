My New Website | Meu Novo Site: fanaro.io

English

Slowly, this website will be deactivated in favor of my new website, fanaro.io. That's a website I've entirely made from scratch — no frameworks — using only HTML, CSS, TS, and Github Pages to host it. In the future, I might add a server and a database backend like Firebase, but, for now GH Pages is more than enough.

The whole experience will be far more minimalist and clean, I hope everyone ends up happier. The new website is completely open source, you can find everything about it here.

The reasons for the changes are based on the fact that WordPress isn't helping me anymore: I've reached a point where I believe the maintenance of something without a framework is smaller than the help WordPress provides initially. The other reasons are performance, cluttering of the webpage with useless WordPress-related stuff and limiting my design creativity.

There's a lot of content here already. In total, there are 107 posts, besides this one. Currently, I've been able to only transcribe about 25% of those, slowly they will all end up there, don't worry, but be patient.



Português

Aos poucos, este site será desativado em prol do meu novo site, fanaro.io. Esse é um site que eu fiz inteiramente sem frameworks, utilizando somente HTML, CSS, TS e Github Pages para hostear. No futuro, eu talvez adicione um servidor e um banco de dados como Firebase, mas, por agora, GH Pages é mais do que suficiente.

Toda a experiência será muito mais minimalist e limpa, espero que todos se sintam mais felizes no final. O novo website é completamente open source — código-fonte aberto —, você pode encontrar tudo que quer sobre ele aqui.

As razões para a mudança são baseadas no fato que o WordPress não está mais ajudando: eu cheguei ao ponto no qual eu acredito que a manutenção de algo sem framework nenhum é menor do que a ajuda que o WordPress pode prover. As outras razões são performance, a confusão desnecessária que o WordPress adiciona à página HTML e a limitação à minha criatividade em termos de design.

Há muito conteúdo aqui já. No total, há 107 posts, sem contar com este. Até o momento, consegui transcrever aproximadamente 25% deles, aos poucos eles acabarão por lá, não se preocupe, mas seja paciente.
